# Ankit Tours & Travels

Premium cab rental, airport transfer, and outstation travel website for **Ankit Tours & Travels** — Mumbai, Maharashtra.

**Live site:** [https://ak-tours.vercel.app](https://ak-tours.vercel.app)

## Features

- Single-page marketing website (`index.html`)
- SEO optimized (meta tags, Open Graph, Twitter Card, JSON-LD)
- Local business schema for Google
- Mobile responsive design
- Quick booking & contact forms
- WhatsApp & call integration

## SEO Files

| File | Purpose |
|------|---------|
| `sitemap.xml` | Helps Google discover pages |
| `robots.txt` | Search engine crawl rules |
| `site.webmanifest` | PWA / mobile app metadata |
| `assets/logo.svg` | Brand logo & favicon |
| `assets/og-image.svg` | Social sharing preview image |

## Google Search Console Setup

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add property: `https://ak-tours.vercel.app`
3. Submit sitemap: `https://ak-tours.vercel.app/sitemap.xml`
4. Request indexing for homepage
